Quick start

After you have ShinySDR open in your web browser (the server will give you the URL when it starts up):

  1. Make sure the “Run” checkbox is checked.

  2. Adjust the “Center frequency” to tune your RF hardware to the band you want to observe.

  3. You can zoom in on the spectrum by scrolling up and down. (Bug: No way to do this without a scrolling input device, particularly not on a touchscreen.)

  4. Click on a signal of interest. This will create a receiver, which will be marked on the spectrum as well as displaying controls. Use the controls to select the appropriate mode (type of demodulation).

    Multiple signals can be received at once by shift-clicking in the spectrum view. To stop, click the X button by the receiver.

Basic concepts

There are sources and receivers. [TODO improve this wording]

A source provides an RF input, and usually corresponds to a hardware device with antenna, tuner, ADC, etc. Which sources are available is set in the configuration file. The source's options, as well as a menu to select the active source if there is more than one, are displayed at the top of the Radio Config section.

A source has a center frequency which determines what range of the radio spectrum is digitized. It is marked by a hairline in the center of the spectrum view.

A receiver selects and demodulates a specific signal within the band received by a source. A receiver has its own channel frequency; whenever the channel frequency is not sufficiently close to the center frequency to be within the bandwidth of the source, it will be inactive.

Receivers are displayed as additional hairlines in the spectrum section and below the source in the Radio Config section. Normally there is only one receiver, but additional receivers can be created and deleted. Note that additional receivers result in additional CPU load on the server.

Tricky bits

Ideally, the UI would be fully discoverable, but right now there are a number of things that are probably not obvious; here's a list of those that haven't already been mentioned:

  • A receiver whose channel frequency is outside of the band around the center frequency will be stopped as long as this is the case.

  • The UI is divided into several sections which can be hidden as desired by clicking on their headers. To show a hidden section, click on its name, which will appear on the right side of the header of the section to the left of it.

  • The large-text frequency fields can be adjusted by scrolling with the mouse cursor as well as by typing digits, and have the following keyboard functions:

    arrow ↑increase selected digit
    arrow ↓decrease selected digit
    arrow ←, backspacemove selection left
    arrow →move selection right
    +change sign to positive
    -change sign to negative
    Zzero selected digit and to the right

    “Z” is a shortcut for entering round numbers: e.g. clicking on the hundred-millions place and typing “144Z” will enter 144,000,000.
